# 后台通用Batch Agent整理框架 - 详细设计文档

## 1. 概述

### 1.1 设计目标

基于参考文档《后台Batch用户意图识别模块.md》的设计理念，实现一个**通用的后台批处理Agent框架**，具备以下特性：

1. **通用性**：支持多种业务场景，通过业务模板定义实体、关系和意图
2. **可配置性**：执行周期、触发条件、业务参数均可配置
3. **动态提示词**：系统提示词支持从MD文件加载或LLM动态生成
4. **系统集成**：与记忆系统、对话上下文、知识图谱深度集成
5. **事件驱动**：检测到实体/关系/意图时触发事件，通知前端UI

### 1.2 核心设计理念

参考文档的核心思想：
- **滑动窗口累积**：维护最近N轮用户输入
- **意图切换触发**：检测到意图转移时立即触发批处理
- **后台静默执行**：零侵入正常对话流程
- **受控词表**：LLM只负责"选择"而非"定义"

本框架将这些理念**泛化为通用模式**，适用于任意业务场景。

### 2.2 核心模块职责

| 模块 | 职责 | 关键特性 |
|------|------|---------|
| **BatchAgentOrchestrator** | 核心编排器，管理整个批处理流程 | 任务调度、生命周期管理、错误处理 |
| **SlidingWindow** | 维护用户输入的滑动窗口 | 窗口大小可配置、支持多种触发策略 |
| **TriggerEngine** | 判定何时触发批处理 | 窗口满、意图转移、时间阈值、自定义条件 |
| **PromptBuilder** | 动态构建系统提示词 | MD文件加载、LLM生成、模板渲染 |
| **BusinessTemplate** | 业务模板定义 | 实体类型、关系类型、意图类型、受控词表 |
| **ExtractionEngine** | 从用户输入中抽取结构化信息 | LLM调用、JSON解析、结果验证 |
| **EventEmitter** | 发射事件通知前端 | 实体检测、关系检测、意图识别事件 |
| **ContextInjector** | 注入对话上下文和记忆信息 | 重要提醒、历史摘要、相关实体 |
| **KnowledgeWriter** | 将抽取结果写入知识图谱 | 实体创建、关系建立、增量合并 |
| **ResultRouter** | 路由处理结果到不同目标 | 知识图谱、文档生成、通知系统 |

## 3. 核心数据结构

### 3.1 配置结构

```rust
/// Batch Agent 配置
#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct BatchAgentConfig {
    /// 执行周期配置
    pub execution: ExecutionConfig,
    
    /// 触发条件配置
    pub trigger: TriggerConfig,
    
    /// 业务参数配置
    pub business: BusinessConfig,
    
    /// 提示词配置
    pub prompt: PromptConfig,
    
    /// 集成配置
    pub integration: IntegrationConfig,
}

/// 执行周期配置
#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct ExecutionConfig {
    /// 执行模式：periodic | event_driven | hybrid
    pub mode: ExecutionMode,
    
    /// 周期执行间隔（秒），仅 periodic 模式
    pub interval_secs: Option<u64>,
    
    /// Cron 表达式，支持更复杂的调度
    pub cron_expression: Option<String>,
    
    /// 最大并发任务数
    pub max_concurrency: usize,
    
    /// 任务超时时间（秒）
    pub timeout_secs: u64,
    
    /// 重试策略
    pub retry_policy: RetryPolicy,
}

/// 触发条件配置
#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct TriggerConfig {
    /// 滑动窗口大小（轮次）
    pub window_size: usize,
    
    /// 时间阈值（秒），超过此时间自动触发
    pub time_threshold_secs: u64,
    
    /// 是否启用意图转移检测
    pub detect_intent_shift: bool,
    
    /// 意图转移检测关键词
